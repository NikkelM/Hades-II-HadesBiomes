To replace a layer with a new image at the same location:
Add the new left and right parts to the "Base" layer.

Copy the left/right part from within GIMP.
Select the layer that should be replaced and use the Edit->Paste As->Paste As Single Layer command.
Remove the old layer


To create a 720p version from a 1080p base:
Create a layer group and move all other layer groups into it
Scale that layer group to a height of 720px using LoHalo interpolation
Select a single layer, crop image to content
Scale image to 1280x720 using LoHalo
